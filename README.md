## 📌 Overview
Docs Mini is a robust document management platform designed for secure storage and structured access. Built using the **MERN stack**, it focuses on fine-grained authorization and high-security API communication to handle sensitive documentation.

## 🚀 Key Features
*   **Advanced RBAC:** Role-Based Access Control ensuring users only see what they are authorized to.
*   **Secure Authentication:** Implementation of JWT with Access and Refresh token logic for persistent, secure sessions.
*   **Cloud Integration:** Seamless document handling and storage.
*   **Responsive UI:** Minimalist, clean interface for document preview and management.

## 🛠️ Technical Stack
| Layer | Technology |
| :--- | :--- |
| **Frontend** | React.js, Tailwind CSS |
| **Backend** | Node.js, Express.js |
| **Database** | MongoDB |
| **Security** | JWT (JSON Web Tokens), Bcrypt |
